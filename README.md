A fast-paced **top-down arcade space shooter** built in **Python using Pygame**.

## ✨ Features

- 💥 **Ultimate Dash Mechanic**  
  A kinetic energy system that unleashes a powerful burst attack.

- 🛡️ **Dynamic Power-ups**  
  Weapon upgrades, shields, and health pickups for evolving gameplay.

- 🖱️ **Precision Mouse Controls**  
  Smooth real-time mouse tracking for fluid ship movement.

- 🔫 **Combat System**  
  - Left Click → Continuous shooting  
  - Right Click → Charged multi-shot burst attack  

- 📈 **Persistent Leaderboard**  
  Local JSON-based Hall of Fame system.

- 🎬 **Arcade VFX**  
  Particle explosions, screen shake, and parallax starfields.

---

## 🕹️ Controls

| Action          | Control                          |
|-----------------|----------------------------------|
| Move Ship       | Mouse Movement                   |
| Shoot           | Left Click                       |
| Burst Attack    | Right Click (hold & release)     |

## ⚙️ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/nova-rift.git
cd nova-rift
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Run the game
```bash
python main.py
```
